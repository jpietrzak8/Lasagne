Lasagne

Barcode-based Inventory Management

Version 0.1

A simple program for inventory management, Lasagne combines the ability to read barcodes with a small spreadsheet. Data is stored in Comma Separated Values (CSV) format, for easy integration with external spreadsheet programs.

Lasagne is a free, open-source program. However, if you enjoy using it, please consider donating a dollar or two. Thanks!
